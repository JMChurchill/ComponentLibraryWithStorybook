# Iconography

For icons we use the material design icons from ReactIcons.

## React Icons

[React Icons Docs](https://react-icons.github.io/react-icons/)

### Material Design Icons

[**License**](https://github.com/google/material-design-icons/blob/master/LICENSE)

[**Project**](https://developers.google.com/fonts/docs/material_icons)

### Install

```bash

```

```jsx

```
